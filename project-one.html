xrite-export

rust   java   WIP

The xrite-export tool is a program designed to make filling in data with a spectrodensitometer much easier.

 

The program works by connecting to the spectro over the xrite DataCatcher program through a data cable.

 

The DataCatcher takes the scanned result of the spectro, and acts as a keyboard wedge, pasting the scanned value into the space the user has selected. The DataCatcher then sends a keyboard input, such as the down arrow key, tab, or enter, and the active field is changed to the next, where the next value can be pasted.
